# 圣多纳释放法APP - 交互设计

## 核心用户流程

### 1. 情绪识别与释放流程
**入口**: 主页面"开始释放"按钮
**步骤**:
1. 情绪选择界面 - 预设情绪标签（焦虑、愤怒、自卑、愧疚、恐惧）+ 自定义输入
2. 身体感受关联 - 用户标记身体不适部位（胸闷、头痛、胃紧等）
3. 释放引导界面 - 语音/文字引导圣多纳四问：
   - "你能允许这个情绪存在吗？"（接纳）
   - "你能放下它吗？"（可能性）
   - "你愿意放下它吗？"（意愿）
   - "什么时候放下？"（行动）
4. 释放完成 - 记录感受，生成成就徽章

### 2. 情绪日记功能
**功能**:
- 日历视图显示释放记录
- 点击日期查看详细释放过程
- 情绪变化趋势图表
- 释放里程碑展示
- 支持文字和图片记录

### 3. 智慧库浏览
**内容分类**:
- 莱斯特的故事（动画/图文）
- 释放法原理（通俗解释）
- 每日名言卡片
- 用户案例分享
- 不同阅读时长（10秒/1分钟/3分钟）

### 4. 每日陪伴功能
**提醒设置**:
- 自定义提醒时间
- 每日小练习推送
- 释放练习建议

## 交互设计原则

### 1. 低门槛操作
- 一键开始释放
- 预设标签快速选择
- 语音引导降低认知负担
- 无需复杂设置

### 2. 温柔反馈
- 释放完成后鼓励语句
- 成就系统正向激励
- 进度可视化但不施压
- 支持随时暂停和退出

### 3. 隐私保护
- 本地数据存储
- 匿名模式
- 明确隐私声明
- 用户完全控制数据

### 4. 渐进式体验
- 新手引导教程
- 功能逐步解锁
- 个性化推荐内容
- 使用习惯学习
